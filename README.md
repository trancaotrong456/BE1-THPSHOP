## 🛠️ Hướng dẫn cài đặt cấu hình API

Dự án này sử dụng Google Cloud API cho tính năng Chatbox. Để chạy được dự án, bạn vui lòng làm theo các bước sau:

1. Truy cập vào https://aistudio.google.com/ để tạo một API Key mới.
2. Mở file `chatbox.php` vừa tạo lên, tìm đến dòng 156 tại 'MÃ_API_KEY_THẬT_CỦA_BẠN_Ở_ĐÂY(https://aistudio.google.com/)' và dán mã API Key của bạn vào.
. Lưu lại và chạy dự án bình thường.
